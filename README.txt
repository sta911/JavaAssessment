For running the application:
	-Navigate to Application folder
	-(double click)run.bat or execute TRG_Assessment.jar from command line
	-Keep all files in the folder

Inside the application you can:
	-Create a new customer account or view the existing accounts.
	-Make purchases.
	-Print all of the available account information.

	-Any new accounts created or purchases made will be saved and will be available for the next run.
	
In the FullProject folder you can find the source code and all the classed created for this application.
The project was implemented using IntelliJ IDE.
